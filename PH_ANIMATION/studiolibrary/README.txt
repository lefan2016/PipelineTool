managing poses and transferring animation in Maya.
Comments, suggestions and bug reports are welcome.

======================================
Installing the Studio Library on Windows.
======================================

# After running the studioLibrary.exe, run the following two lines in the PYTHON script editor
import studiolibrary
studiolibrary.main()


======================================
Installing the Studio Library on OSX (video).
======================================

1. Unzip and move the studiolibrary folder to the maya scripts path...
*Replace USERNAME with your user name.
/Users/USERNAME/Library/Preferences/Autodesk/maya/scripts

2. Run the following code in the Python script editor.
import studiolibrary
studiolibrary.main()
